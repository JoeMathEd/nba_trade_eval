## Overview
* Runs a Shiny app that allows users to explore the impact of NBA trades on team performance
* Users view each team need and reccomended player to trade for
* Users can adjust sliders to weight different factors such as Offense, 3pt shooting, and Defense to see how it affects the recommended trade
* Comparison tab of reccomended trades vs actual trades at the NBA trade deadline
* Tab showing distrubutions of teams for each factor (offense, 3pt shooting, defense)

## Data Collection
* Lineup data was scraped from Databallr.com and cleaned using R
* Player data collected from ballr package in R
* Two datasets were merged together to create a comprehensive dataset for analysis (data_merge.R)
